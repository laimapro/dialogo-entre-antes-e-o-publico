Você está em: Acessibilidade neste DVD

- Acesse o menu  principal, teclando a letra L.
- Acesse  o link acessibilidade neste  DVD, teclando alt1.
- Acesse  o link  artigos em áudio,  teclando  alt 2, pressione espaço ou enter para  ouvir os artigos.
- Acesse  o link artigos em Libras, teclando alt 3, pressione espaço ou enter para ler os artigos em Libras.
- Acesse o link artigos em texto, teclando alt 4,  pressione espaço ou enter para ler os artigos em pdf.
- Acesse o link apresentação, teclando alt 5.
- Acesse o link editorial, teclando  alt 6.
- Para baixar  o  artigo desejado, pressione   espaço ou enter  e depois tecle alt b, então pressione enter ou espaço   novamente para  escolher abrir o arquivo ou  salvá-lo em uma pasta.
- Para recarregar um arquivo de áudio ou de vídeo, pressione F5.
- Para interromper o áudio ou vídeo, pressione ESC.
- Para acessar o menu principal agora, ou a qualquer momento, pressione uma das teclas de atalho indicadas.
